# APACHE II Calculator

## Objetivo

Calcular o APACHE II para estratificação de gravidade em UTI.

## Rodar localmente

```bash
npm install
npm run dev
```

## Testes

```bash
npm run test
```

## Build

```bash
npm run build
```

## Deploy

```bash
vercel deploy
```
